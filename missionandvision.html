Coding Club

Central Institute of Technology, Kokrajhar

Mission and Vision of the Club

Objectives of Coding Club


  1. To bring up the coding culture in our institute
  2. To acquaint new comers to programming.
  3. To discuss programming concepts using various languages such as C/C++, Java, Python etc.
  4. To get started with competitive programming.
  5. To compete in various coding contests.




Other Objectives of Coding Club


  1. Undertake institute projects
  2. Organize various competitions
  3. Collaborate with other institutions
  4. Helping platform for beginners




Focussed Areas/Topics


  1. Programming Languages (C/C++/Java/Python)
  2. Competitive programming
  3. Scripting languages
  4. Data Structures & Algorithms and Design and Analysis of Algorithms
  5. Software Development




For Whom?

Anyone and Everyone who


  1. Possesses keen interest in coding
  2. Wants to enter into Competitive Programming
  3. Has interest in Software Development
  4. Wants to explore various domains in computer science
